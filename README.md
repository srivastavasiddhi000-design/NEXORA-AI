# NEXORA AI 🤖

A Multi AI Agent System built using Generative AI.

## 🚀 Features

- 🔍 Research Agent
- 📚 Knowledge Agent (PDF RAG)
- 🚀 Planner Agent
- 🎤 Voice Input
- 💬 Conversational AI Chat
- 🧠 Context-aware AI responses


## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- RAG (Retrieval Augmented Generation)
- LLM
- Speech Recognition


## 📂 Project Structure 

NEXORA-AI
│
├── app.py
├── main.py
├── rag.py
├── router.py
├── summary_agent.py
├── quiz_agent.py
├── vectorstore.py
└── requirements.txt



## ▶️ Run Locally

Install dependencies:


pip install -r requirements.txt


Start application:


streamlit run app.py



## 💡 About

NEXORA AI is a multi-agent AI assistant that combines research,
document understanding, and intelligent task planning into one system.
